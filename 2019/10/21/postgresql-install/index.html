服务端

  • 创建数据库集群
initdb -D "K:\PostgreSQL\data"
  • 启动数据库服务器
pg_ctl -D ^"K^:^\PostgreSQL^\data^" -l logfile start
  • 注册Windows服务
pg_ctl register -N PostgreSQL -D "K:\PostgreSQL\data"
  • 注销Windows服务
pg_ctl unregister -N PostgreSQL
